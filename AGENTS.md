## 🎯 Project Vision
Create a premium developer portfolio for "BERNY DEV", a Full Stack Developer focused on AI-driven experiences.

The site must feel:
- Modern
- Minimalist
- High-end
- Visually impactful (WOW effect)
- Technically solid

### Colors
- Background: #0A0A0F

- Primary Neon: #A855F7
- Secondary Violet: #9333EA
- Glow: #C084FC
- Text: #E5E7EB

### Style
- Dark mode only
- Soft glow effects (NOT overused)
- Rounded corners (xl / 2xl)
- Clean spacing
- Subtle glassmorphism
- Smooth animations

### Typography
- Primary: Inter
- Headings: Sora (optional)
